# 小米汽车体系运营部抽奖系统 🎉

年会抽奖专用系统，支持炫酷3D球体动画效果。

</div>

一个为小米汽车体系运营部定制的年会抽奖应用，具有炫酷的3D球体效果，支持奖品、人员、界面等配置。

## 功能特性

- 🕍 炫酷3D球体动画效果
- 💾 本地数据持久化存储
- 🎁 灵活的奖品奖项配置
- 👱 便捷的人员名单管理
- 🎼 背景音乐播放
- 🖼️ Excel导入导出功能
- 🎈 支持临时抽奖
- 🧨 多语言支持
- 🍃 自定义背景图片

## 使用要求

请使用PC端最新版Chrome或Edge浏览器访问。

## 配置说明

### 人员配置

在人员配置管理界面下载Excel模板，填写人员信息后导入。

### 奖项配置

在奖项配置管理界面添加奖项，自定义名称、抽取人数、参与范围等。

### 界面配置

支持自定义标题、卡片颜色、列数、图案等视觉元素。

### 图片和音乐

支持上传自定义图片和音乐，数据存储于浏览器本地IndexDB。

## 技术栈

- Vue 3
- Three.js
- IndexDB
- Pinia
- DaisyUI

## 开发说明

安装依赖：

```bash
pnpm install
```

开发运行：

```bash
pnpm dev
```

打包构建：

```bash
pnpm build
```
